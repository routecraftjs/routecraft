Logo options  ·  v0.1  ·  pick one

Eight marks. One belongs.

Each concept uses only the brand palette (paper, ink, cobalt-500) and reads at favicon size. Pick one, refine it once, then use it everywhere: favicon, profile photo, app icon, t-shirt.

Right-click → Inspect element on any logo to copy the raw SVG markup. Every mark is one path/rect/text element so it scales infinitely and stays under 1 KB.
