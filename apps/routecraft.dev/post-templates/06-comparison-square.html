Routecraft
Bounded by design

Hands, not keys.

01

The keyring

Hand the agent raw tools and hope. 

// Typical agent SDK
agent({
  tools: [
    bashTool(),
    httpTool({ allowAll: true }),
    dbQueryTool(conn),
  ],
})

One bad prompt away from an incident.

02

The hands you authored

Named capabilities. Nothing else. 

// A capability that drives the agent
craft()
  .id('triage')
  .from(mail('INBOX'))
  .to(agent({
    tools: tools(['publishBrief']),
  }))

The capability is the boundary.
