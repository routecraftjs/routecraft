Filter at the source.

One pipeline step. The whole inbox, narrowed to what matters. Less context, faster runs, fewer mistakes.

Every primitive an agent app needs.

triage.ts filter 

craft()
  .id('triage-inbox')
  .from(mail('INBOX'))
  .filter(({ from }) =>
    /@company\.com$/.test(from))
  .transform(redact)
  .to(agent('triage'))
